Cloud Monitoring & Alert System

Java | AWS EC2 | CloudWatch | SNS | HTML | CSS | JS

A full-stack cloud monitoring dashboard that tracks server performance in real-time using AWS services. Features threshold-based alerting and a responsive visualization frontend.

  • Developed Java backend integrated with AWS EC2, CloudWatch, and SNS for live metric tracking.
  • Built responsive frontend with HTML/CSS/JS to visualize CPU, memory, and network data.

OS Resource Monitoring Tool

C | Linux | Ubuntu | GCC | GDB | System Programming

A real-time Linux resource monitoring tool written in C, reading directly from kernel files to track CPU, memory, and disk I/O with no external dependencies.

  • Parsed /proc/stat, /proc/meminfo, and /proc/diskstats for live performance data.
  • Modular architecture with configurable polling intervals.
  • Threshold alerts for high CPU and memory usage rendered directly in terminal.
  • Compiled & debugged with GCC and GDB on Ubuntu environment.
